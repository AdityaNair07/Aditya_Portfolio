Basic Text Editor

The Basic Text Editor provides user to edit/format the text that they enter. Formattings like bold, italic, underline, strikethrough, etc. can be used in the editor. As well as font formatting like changing forecolor and highlighting the text. User has the functionality to add ordered and unordered list as well that allows them to create organised set of words. Basic text alignments, font size, font family, etc. can be used. The functionality to add hyperlinks to a text is also provided in this project.

Live Language Translator

The Live Language Translator is an HTML/CSS JS based project used to translate the user input on the go. The user can choose from a variety of languages (the language to write and language to translate). The functionality to hear what is to be translated and the translated text makes the translator user friendly. Also copying the text to the clipboard makes the process easy. The project has a simple UI and easy navigation with Instructions given at the bottom of the page. The project makes use of the language translation api which helps in converting the input text to desired translated text.

Recipe Finder

Are you a foodie or an enthusiastic cook? Then The Recipe Finder will help you get recipes of almost every cuisine you can think of. It is an HTML/CSS Js based project which is used to find recipe with the help of api. The UI is simple and user friendly navigation keeps the page easy to access. The user can search the dish in the search bar and the api will fetch the results which are then displayed on the screen. Each recipe comes with a view recipe button which leads the user to the article/blog which explains the recipe.

To-Do List

When you have to keep track of your daily important tasks and don't know how... That's where to-do list comes to the rescue. The To-Do List is a React Js based project. This helps the user to keep track of the important tasks that are to be performed in a particular day. With add and delete functionality user can add and delete tasks on the list.
